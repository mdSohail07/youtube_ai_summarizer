# 🎥 YouTube AI Summarizer

An AI-powered YouTube video summarizer built with Python and Streamlit.

## ✨ Features

- 🔗 Summarize YouTube videos
- 🤖 AI-generated concise summaries
- ⚡ Fast and easy to use
- 🎨 Clean Streamlit interface

## 🛠️ Tech Stack

- Python
- Streamlit
- OpenRouter / LLM
- Requests

## Installation

```bash
git clone https://github.com/mdSohail07/youtube_ai_summarizer.git
cd youtube_ai_summarizer

pip install -r requirements.txt
streamlit run app.py
```

## Environment Variables

Create a `.env` file:

```
GOOGLE_API_KEY=your_api_key
```
